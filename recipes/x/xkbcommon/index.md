# xkbcommon

## Version
1.5.0

## Description
xkbcommon is a keyboard handling library providing a portable way to process key codes and compose keymaps. It is the modern replacement for X11 keyboard handling and is used by Wayland compositors.

## Dependencies
- libxcb (optional)
