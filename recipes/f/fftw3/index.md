# fftw3

## Version
3.3.10

## Description
FFTW is a free, fast C library for computing the Discrete Fourier Transform (DFT). It is one of the fastest FFT implementations available, supporting 1D, 2D, and 3D transforms, and both real and complex data.

## Dependencies
- None (pure C library)
